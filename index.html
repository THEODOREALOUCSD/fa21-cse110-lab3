What is a boxing timer?

A boxing timer is a timer that sets a round time, how many rounds, and breaks between rounds without the user having to manually set the clock between rounds.

What should a good boxing timer have?

A good boxing timer should have a simple to use interface where setting the rounds, round time, and break should be easy to set up. This timer should be as simple as possible.

How should we implement the app?

  1. Learn the tools for app development
  2. Use the tools to implement the core features
  3. Make the app look simple and presentable.
